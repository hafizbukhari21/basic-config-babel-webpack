# Simple Webpack


## Documentation
- Clone from this repository then type ___npm install___ for first time
- type ___npm start-hafiz___ then ___npm run dev-hafiz___ for running from node server

```bash
npm install
```

```bash
npm start-hafiz
```

#### Start Server mode
```bash
npm run dev-hafiz
```
